# The Scratchpad: Go -> Python -> Cyber

Starting completely from zero. 

I'm ignoring the standard web dev tutorials everyone else follows. The plan is to get into cybersecurity, but I want to build the actual foundation first. 

The route: Go -> Python -> Cyber.

I'm doing Go first because I want to actually understand memory, strict typing, and how systems work under the hood. Once the logic is solid, I'll switch to Python for faster scripting, and then finally start breaking things.

This repo isn't a polished portfolio. It's just a raw dump of my daily practice. The code will be messy, it will break, and I will probably do things the wrong way while I learn. 

### Folders
- `/attempts` - code that throws errors and my rough experiments
- `/working` - scripts that actually compile and run successfully
